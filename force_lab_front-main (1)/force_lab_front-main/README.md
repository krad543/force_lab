**FORCE LAB (Frontend)** — это клиентская часть платформы для управления спортивной деятельностью.

## Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/asyakhar/force_lab_front

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

```

Приложение будет доступно по адресу: `http://localhost:5173`
